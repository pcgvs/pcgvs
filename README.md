## ‚ö° Quickstart

Create another conda environment: 

```bash
conda create --name pcgvs python=3.9
```

Install the package locally:

```bash
pip install -e .
```

Now you can use the package in your scripts! For example:

```python
from pcgvs.extraction import TubeExtraction

...
```
